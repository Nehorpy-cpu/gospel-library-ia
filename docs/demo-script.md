# Demo script

## Objetivo

Mostrar Gospel Library IA Beta con datos reales, privacidad, fallback textual y flujo de estudio personal.

## Flujo

1. Abrir `http://localhost:3000/beta`.
2. Iniciar sesion como usuario beta aprobado.
3. Completar onboarding con perfil de estudio e idioma.
4. Ir a `/search` y buscar `Mosiah 14:11`.
5. Revisar resultados y confirmar que las fuentes aparecen con titulo, autor y URL.
6. Crear un espacio en `/study/new`.
7. Abrir el workspace y agregar un pensamiento como nota.
8. Abrir un documento, seleccionar una cita y guardarla en el workspace.
9. Crear un post-it con una aplicacion personal.
10. Usar filtros de fuentes y revisar resultados relacionados.
11. Ir a `/talk-builder`, construir un bosquejo y revisar citas usadas.
12. Exportar notas/citas en Markdown o PDF.
13. Enviar feedback desde el boton flotante.
14. Entrar como admin y abrir `/admin`.
15. Revisar feedback, usuarios beta, costos IA, Qdrant, PostgreSQL y fuentes.

## Mensaje doctrinal durante demo

Recordar que la app no reemplaza las escrituras, fuentes oficiales, lideres autorizados ni revelacion personal. La IA debe ayudar a encontrar y organizar fuentes, no declarar doctrina nueva.
